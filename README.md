# LNX — The Worst Programming Language Ever

> ⚠️ **Status:** Half-baked, intentionally. That's the point.

A language carefully designed to be as frustrating and illogical as possible. If you're here, you either love pain or lost a bet.

## Features (or Anti-Features)

- **Absurd Boolean System**: `definitely_true`, `kinda_sorta`, `nope`
- **Mixed Syntax Rules**: Indentation-based AND bracket-based, inconsistently enforced
- **Confusing Operators**: `+` means multiplication, `-` means addition
- **Pointless Variable Rules**: Names must be ≥15 chars, contain a number at position 4, and an uppercase at position 12
- **One Error Message**: Everything fails with `lol nope`
- **Required Script Starters**: Scripts must begin with nonsense declarations

## Why?

Some people build languages to change the world. I built one to watch it burn.
