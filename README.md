# The Worst Programming Language Ever

## Introduction
Welcome to **The Worst Programming Language Ever**, a language carefully designed to be as frustrating and illogical as possible. If you're here, you either love pain or lost a bet.

## Features (or Anti-Features)
This language includes a variety of **deliberately bad** features:
- **Absurd Boolean System**: `definitely_true`, `kinda_sorta`, and other nonsense.
- **Mixed Syntax Rules**: Both indentation-based and bracket-based syntax, inconsistently enforced.
- **Confusing Operators**: `+` means multiplication, `-` means addition, etc.
- **Pointless Variable Rules**: Variable names must be at least 15 characters, contain a number at position 4, and an uppercase letter at position 12.
- **Weird String Rules**: Quote style depends on string length.
- **One Error Message for Everything**: Every mistake results in `lol nope`.
- **Required Script Starters**: Scripts must begin with nonsense declarations.

For full details, see `feature-wiki.md`.

## Installation
You'll need **Python 3.8+** to run this disaster.
```sh
pip install -r requirements.txt
```

## Usage
Run the interactive REPL:
```sh
python repl.py
```
Or execute a script:
```sh
python main.py my_script.wpl
```
